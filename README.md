# IT Support Tool

Автор: Бурсук Станіслав  Група: ІН-22
Клієнт-серверна система для подачі заявок у техпідтримку.


## Функціонал
- Автоматичний збір даних (IP, Host).
- Діагностика мережі.
- Збереження в PostgreSQL.
- Резервне збереження в файл.

## Запуск
python report_tool.py

## Інструкція користувача
1. Знайдінь ярлик програми та відкрийте її
2. Опишіть проблему яка виникла у області "Опис проблеми"
3. Оберіть з випадаючого списку систему де саме виникла проблема
4. Якщо потрібно додати скрішнот поставте галочку (проблему має бути видно на екрані)
5. Натисніть кнопку "Відправити заявку"
